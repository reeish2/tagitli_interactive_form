מבוא ל-HTML

HTML (ראשי תיבות של HyperText Markup Language) היא השפה הבסיסית שבה משתמשים ליצירת דפי אינטרנט.
באמצעות HTML, ניתן להציג תכנים כמו טקסט, תמונות, קישורים ועוד.


תגיות HTML

HTML פועלת על ידי שימוש בתגיות (tags) שמסמנות חלקים שונים של התוכן.
כל תגית מציינת סוג מסוים של תוכן, כמו כותרת, פסקה, קישור או תמונה.
דפדפן האינטרנט קורא את התגיות הללו ומציג את התוכן בהתאם למבנה שהוגדר.


מבנה בסיסי של דף HTML

מבנה של דף HTML טיפוסי כולל מספר חלקים בסיסיים:

  • הגדרת סוג המסמך: מציינת לדפדפן שהמסמך הוא דף HTML.
  • תגית שורש: מכילה את כל התוכן של הדף.
  • חלק הראש (head): כולל מידע על הדף כמו כותרת, קידוד ושורות סגנון.
  • חלק הגוף (body): מכיל את התוכן שמוצג למשתמש, כגון טקסטים, תמונות וקישורים.
