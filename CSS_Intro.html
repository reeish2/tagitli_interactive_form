מבוא ל-CSS

CSS (ראשי תיבות של Cascading Style Sheets) הוא שפה המשמשת לעיצוב והצגת דפי HTML.
HTML מספק את מבנה הדף, ואילו CSS מאפשר להגדיר את המראה החיצוני שלו.


למה משמש CSS?

CSS מאפשר לעצב דפי אינטרנט באמצעות הגדרת צבעים, גופנים, מרווחים, גבולות, ועוד.
כך ניתן להפריד בין תוכן (HTML) לעיצוב (CSS), מה שמקל על תחזוקה ועיצוב עקבי של האתר.


מבנה קובץ CSS

קובץ CSS מורכב מחוקים.
כל חוק כולל בורר (selector) והצהרה (declaration).
ההצהרה כוללת תכונה (property) וערך (value).


סוגי בוררים

  • בורר סוג (Type Selector): בוחר אלמנטים לפי שם התגית.
  • בורר מזהה (ID Selector): בוחר אלמנטים לפי מזהה ייחודי.
  • בורר מחלקה (Class Selector): בוחר אלמנטים לפי מחלקה.

שימושים נוספים ב-CSS

  • יצירת עיצובים במצבים ספציפיים או על חלקים מסוימים של האלמנט (למשל :hover).
  • שיטות לעיצוב מערכי אלמנטים בדף.
  • אנימציות ומעברים: הוספת תנועה ואפקטים לאלמנטים.
